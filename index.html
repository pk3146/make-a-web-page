Inline vs Block

br is inline and p is block. br is simply ending a line, p tag works differently and is actually making an invisible box. The invisible box can have height and width. Note, the p element is the only block element we've learned so far.
Span is inline and div is block.

Basic HTML Vocabulary


Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag (p for example) and a closing tag, (forward slash p). Some tags (called "void" tags) do not require a closing tag (like the br tag).
Element: An HTML element refers to everything within a set of opening and closing tags.
Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Udacity URL, you would write a href="www.udacity.com" [note removed tags to view in notes]

Web Pages, Documents and Structure

One of the key concepts to remember is that although HTML and CSS are both languages, they are quite different. Think of HTML like the structure of a home (i.e the foundation and walls), and CSS is the style, or decoration within the home. Every webpage has a structure. HTML elements are either inline or block. If they are block, the structure can be broken down into invisible squares. The element div allows you to add structure, sort of like drawing an invisible box around similar chunks. When writing code, it is important to use logical arrangement and avoid unneccessary repetition. Break the html code into logical sections within divs, such as heading or class and then write associated css code to style the sections.

Random but Important Facts

Google Chrome has a web editor tool that allows you to see the structure and code of a page. The structure is similar to a tree, with a main trunk and branches. The treelike structure is due to the fact that HTML elements can have elements nested within them.

Sublime allows you to view both html and css side by side. Remember they are unique files, and to save changes in each file. To view changes made in the html in Sublime in a web browser, save in Sublime and then hit refresh on the web browser.

The Box Model

HTML elements are boxes and each box has four components which are Margin, Border, Padding and Content. Box sizes don't have to be in pixels, they can be expressed in percentages also. If set as a percentage, the size will change depending on how big the screen is. Or, can set the box sizing attribute to border box for each element. CSS is very powerful in allowing element positioning, but it is very complex to understand. Flexbox is one method (see instructor notes). Divs are block elements (not inline) so they take up the entire width of a page by default. By adding rules, can override this behavior and let the divs appear side by side.'
